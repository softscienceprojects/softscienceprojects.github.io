Curatorial

Demo is a scaled-down version with only 50 artworks

An artwork-searching app with functionality to create exhibitions, save artworks and follow other users

Features a Rails API backend with custom routes. Postgres database seeded by scraping Harvard Art Museum API then images analysed by Google Vision API. Local database scraped over 2000 artworks.

Front-end built with React, including nested routes, search functionality and front-end validation. Custom CSS for styling.
